🔘 Experiment: Push Button Interface with LED (STM32)
📌 Project Overview
This experiment demonstrates how to interface a push button as digital input and control an LED as digital output using an STM32 microcontroller.
It illustrates basic sensor–actuator control, where the microcontroller reads input from a button and drives an LED accordingly. 

🎯 Aim- To interface a push button as digital input with an LED as digital output for simple sensor-actuator control.
